# Problem #1
What is the maximum number of consecutive zeros in a given binary number 
that are surrounded by ones?

### Examples:

| dez   |                 bin | result |
|-------|--------------------:|-------:|
| 0     |                   0 |      0 |
| 1     |                   1 |      0 |
| 2     |                  10 |      0 |
| 3     |                  11 |      0 |
| 8     |                1000 |      0 |
| 15    |                1111 |      0 |
| 5     |                 101 |      1 |
| 9     |                1001 |      2 |
| 157   |           1001 1101 |      2 |
| 165   |           1010 0101 |      2 |
| 261   |         1 0000 0101 |      5 |
| 37133 | 1001 0001 0000 1101 |      4 |
